算法工具函数汇总

发表于 2021-09-13 更新于 2022-04-10 分类于 数据结构与算法
本文字数： 1.1k 阅读时长 ≈ 1 分钟

将刷算法题中的遇到和用到的工具函数汇总。

一、数学

判断一个数是否是2的n次方

    public boolean isPowerOfTwo(int n) {
        return n > 0 && (n & (n - 1)) == 0;
    }

求最大公约数和最小公倍数

    //最大公约数
    public int gcd(int x,int y){
        return y > 0 ? gcd(y , x % y) : x;
    }
    //最小公倍数
    public int lcm(int x,int y){
            return x / gdc(x, y) * y;
    }

二、数组操作

复制数组内容

    //复制数组
    System.arraycopy(fromArr,0,toArr,0,len);

排序二维数组

    Arrays.sort(arr, (o1, o2) -> Integer.compare(o1[0], o2[0]));

List<int[ ]>转换为int[ ][ ]

    arr.toArray(new int[arr.size()][]);

List<String>转换为String[]

    arr.toArray(new String[arr.size()]);

三、字符串操作

split()分割字符串注意

1、split方法转化字符串为数组：

   String[] strPicArr = s.split("*");
   //报错：
   //java.util.[regex](https://so.csdn.net/so/search?q=regex&spm=1001.2101.3001.7020).PatternSyntaxException: Dangling meta character '*' near index 0

因为+、*、|、\等符号在正则表达示中有相应的不同意义，所以在使用时要进行转义处理。

这里同样可以用

   String[] strPicArr = map.get("hw_pic").toString().split("[*]");

String类型转换大小写

    s = s.toLowerCase();//转小写
    s = s.toUpperCase();//转大写

Character类型转换大小写

    ch = Character.toLowerCase(ch);
    ch = Character.toUpperCase(ch);

四、位运算

Java位运算相加(C++和Python需要额外处理负数)

    public int add(int a, int b) {
        return b == 0 ? a : add(a ^ b, (a & b) << 1);
    }
