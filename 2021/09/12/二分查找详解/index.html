寻找一个数（基本的二分搜索）

class Solution {
    public int search(int[] nums, int target) {
        int left = 0;
        int right = nums.length - 1;    //这里-1时，while循环条件得加上=
        while(left <= right){            //注意
            int mid = left + (right - left) / 2; //防止了 left 和 right 太大直接相加导致溢出
            if(target == nums[mid]){
                return mid;
            }else if(target < nums[mid]){
                //如果right初始化为nums.length - 1，这里应该为mid-1;
                right = mid - 1;        
            }else if(target > nums[mid]){
                left = mid + 1;            //注意
            }
        }
        return -1;
    }
}

寻找左侧边界的二分搜索

public int search(int[] nums, int target) {
        int left = 0,right = nums.length;
        while(left < right){
            int mid = left + (right - left) / 2;
            //寻找左、右侧边界的二分搜索要注意
            //收紧边界时的条件很重要，
            //这里right = mid ，并且循环结束的条件是 right = left
            //所以循环结束是 right = mid = left
            if(target == nums[mid]){
                right = mid;
            }else if(target < nums[mid]){
                right = mid;
            }else if(target > nums[mid]){
                left = mid + 1;
            }
        }
        if(left >= nums.length || nums[left] != target) return -1;
        return left;
    }

寻找右侧边界的二分搜索

public int search(int[] nums, int target) {
        int left = 0,right = nums.length;
        while(left < right){
            int mid = left + (right - left) / 2;
            if(target == nums[mid]){
                //收紧边界时的条件很重要，
                //这里left = mid + 1;并且循环结束的条件是 right = left
                //所以循环结束是 mid = left - 1 = right - 1；
                left = mid + 1;
            }else if(target < nums[mid]){
                right = mid;
            }else if(target > nums[mid]){
                left = mid + 1;
            }
        }
        if(right - 1 < 0 || nums[right - 1] != target) return -1;
        return right -1;
    }

LeetCode 153. 寻找旋转排序数组中的最小值

思路：根据左值，中值，右值的大小关系分类讨论.

其中符合数组规律的大小关系如下:

​ 右 中值 < 右值 ， 缩小右边界

​ 中

左

​ 中 中值 > 右值 ， 缩小左边界

左

​ 右

左 中值 < 右值 , 缩小右边界

​ 右

​ 中

class Solution {
    public int findMin(int[] nums) {
        int left = 0,right = nums.length - 1;
        while(left < right){
            int mid = (left + right) / 2;
            if(nums[mid] < nums[right]){
                right = mid;
            }else{
                left = mid + 1;
            }
        }
        return nums[left];
    }
}

LeetCode 162. 寻找峰值

根据左右指针计算中间位置 m，并比较 m 与 m+1 的值，如果 m 较大，则左侧存在峰值，r = m，如果 m + 1 较大，则右侧存在峰值，l = m + 1.

class Solution {
    public int findPeakElement(int[] nums) {
        int left = 0,right = nums.length - 1;
        while(left < right){
            int mid = (left + right) / 2;
            if(nums[mid] > nums[mid + 1]){
                right = mid;
            }else{
                left = mid + 1;
            }
        }
        return left;

    }
}
