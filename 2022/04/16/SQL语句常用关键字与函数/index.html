SQL语句常用关键字与函数

发表于 2022-04-16 更新于 2022-04-19 分类于 MySQL
本文字数： 3k 阅读时长 ≈ 3 分钟

在leetcode中刷MySQL题目时遇到很多平时没用过的函数，记录一下常用关键字与函数

关键字

查询结果组合关键字（union，union all）

MySQL union 用于把来自多个select 语句的结果组合到一个结果集合中。语法为：

    select  column,......from table1
    union [all]
    select  column,...... from table2

union all使用案例1：LeetCode1965. 丢失信息的雇员

    select employee_id
    from (
        select employee_id from employees
        union all
        select employee_id from  salaries
    ) ans
    group by employee_id
    having count(employee_id) = 1
    order by employee_id

union all 使用案例2：LeetCode1795. 每个产品在不同商店的价格

    select product_id,'store1' as store,store1 as price from products where store1 is not null
    union all
    select product_id,'store2' as store,store2 as price from products where store2 is not null
    union all
    select product_id,'store3' as store,store3 as price from products where store3 is not null

union 与 union all 的区别

当使用union 时，MySQL 会把结果集中重复的记录删掉，而使用union all ，MySQL 会把所有的记录返回，且效率高于union 。

正则筛选关键字（regexp）

使用案例LeetCode1527. 患某种疾病的患者

    select * 
    from Patients
    where conditions regexp '^DIAB1|\\sDIAB1'

去重关键字（distinct）

使用案例LeetCode 176. 第二高的薪水

    select (
        select distinct salary from Employee 
        order by salary desc
        limit 1,1
    ) as SecondHighestSalary
扩展

distinct 是先排序结果然后去除相邻重复的结果，因此distinct结果是排序过的

条件查询关键字(case when)

使用案例1：LeetCode1873. 计算特殊奖金

    select employee_id,(
        case when employee_id%2 = 1 and  name not like 'M%' then salary else 0 end
        ) as bonus 
    from Employees 
    order by employee_id

使用案例2：LeetCode627. 变更性别

    #写法一
    update Salary
    set sex = (case when sex = 'f' then 'm' else 'f' end)
    #写法二
    update Salary
    set sex = (case sex when 'f' then 'm' else 'f' end)

使用案例3：LeetCode608. 树节点

    select id,(
        case
        when p_id is null then 'Root' 
        when id in (select p_id from tree) then 'Inner'
        else 'Leaf' 
        end
    ) as type
    from tree

分组关键字（group by）

使用案例196. 删除重复的电子邮箱

    DELETE from Person 
    Where Id not in (
        Select Id 
        From(Select MIN(Id) as id From Person Group by Email) t
    )
    #同一题不同解法
    DELETE p1
    from person p1,person p2
    where p1.email = p2.email and p1.id > p2.id

分页查询关键字（limit）

用法：limit start,pageSize

​ start：表示从第start个数据开始查询

​ pageSize：表示查询的数据长度为pageSize

使用案例LeetCode 176. 第二高的薪水

    select (
        select distinct salary from Employee 
        order by salary desc
        limit 1,1
    ) as SecondHighestSalary

函数

字符串分割函数（left(),right(),substring()）

用法：left (str,len) , right (str,len) , substring(str,start,len)

​ 第一个参数str表示处理的字符串

​ left (str,len) ：表示截取str靠左的len个字符

​ right (str,len) ：表示截取str靠右的len个字符

​ substring (str,start,len) ：与Java语法中一致，不做解释(注意：**MySQL中字符串字符下标以1开始**)

使用案例1667. 修复表中的名字

    select user_id,concat(upper(left(name,1)),lower(substring(name,2,1))) as name
    from Users
    order by user_id

字符串合并函数（concat(),group_concat()）

用法：concat( str1,str2,str3, … … ) , group_concat()

使用案例1484. 按日期分组销售产品

    select sell_date,count(distinct product) as num_sold ,group_concat(distinct product) as products
    from Activities 
    group by sell_date
    order by sell_date

条件判断函数（if()）

使用案例1：LeetCode1873. 计算特殊奖金

    select employee_id,if(employee_id%2 = 1 && left(name,1) != 'M',salary,0) as bonus 
    from Employees 
    order by employee_id

使用案例2：LeetCode627. 变更性别

    update Salary
    set sex = if(sex = 'm','f','m')

使用案例3：LeetCode608. 树节点

    #根据提交结果来看，`is null`比`isnull()` 效率更高一点。因此这里使用is null判断更好
    select id,
        if(isnull(p_id),'Root',if(id in (select p_id from tree),'Inner','Leaf')) as type
    from tree

判空函数（isnull()）

与is null是一样的效果，反正就是在MySQL中不能使用=null判断是否为null

根据LeetCode608. 树节点提交结果来看，is null比isnull() 效率更高一点。
